# 비급여·랜딩 치료재료 관리 웹앱 V1

## 실행
1. `index.html`을 크롬/엣지에서 엽니다.
2. 데이터는 브라우저 localStorage에 저장됩니다.
3. 상단 `백업 JSON`으로 수시 백업하세요.

## 포함 기능
- 수술별 사용 입력
- 원장님별/수술종류별 추천 비급여
- 추천 비급여 미선택 확인 팝업
- 비급여 제한 환자 사용 확인 팝업
- 당일 현황 및 수정
- 구매제품 입고 수량 입력
- DBM/ANCHOR 랜딩 입고 주간 체크판
- 제품 설정/재고 경고
- 기간별 통계 및 CSV 다운로드

## 주의
- 병원 서버/OCS와 직접 연결하지 않는 독립형 1차 프로토타입입니다.
- 여러 태블릿에서 동시에 쓰려면 2차에서 서버/Firebase/Supabase 같은 공용 DB 연결이 필요합니다.
